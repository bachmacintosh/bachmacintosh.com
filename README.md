## Development

Install Dependencies

```shell
$ npm install
```

Available Commands

```shell
# run dev server
$ npm run dev

# Build an optimized Next.js App
$ npm run build

# Export Static HTML
$ npm run export

# Combine build and export for deploying to Netlify
$ npm run export:netlify

# Start serving using the Next.js server
$ npm run start

# run ESLint
$ npm run lint

# run Cypress in-browser
# Terminal 1
$ npm run build
$ npm run start

# Terminal 2
$ npm run cypress
```
